\Abstract{Palavras Chave}{Arqueologia, Deep learning, LiDAR, Unet, YOLOv7}
         {Resumo}{\selectlanguage{portuguese}Esta dissertação foi desenvolvida no ambito do projetio ODYSEY. O projeto tem como objetivo desenvolver uma plataforma destinada a arqueólogos que contenha informação geografica.
Dentro do âmbito do projeto, esta dissertacao foca-se no uso de algortimos de deep learning, com o objetivo de identificar sitios arqueologicos, atrves de imagens provenientes de dados recolhidos com LiDAR. A zona escolhida para o projeto foi o Alto Minho e os objetos de estudo foram mamoas e castros. Para alcancar esse objetivo foi preciso processar essas imagens, de modo a criar o dataset, assim como criar metodos de data augmentation, depois disso foram usados dois algoritmos de deep learning, um de object detection e um de semantic segmentation, YOLOv7 e Unet respetivamente. No fim foi feito inferencias nas imagens providas de modo a tentar encontrar novos sitios arqueologicos ainda nao anotados.}

\Abstract{Keywords}{Archaeology, Deep learning, LiDAR, Unet, YOLOv7.}
         {Abstract}{\selectlanguage{english}This dissertation was developed within the ODYSEY project. The project aims to develop a platform for archeologists that contains geographic information.
Within the ambit of the project, this dissertation focuses on the use of deep learning algorithms, with the objective of identifying archaeological sites, through images coming from data collected with LiDAR. The area chosen for the project was Alto Minho and the objects of study were tumuli and hillforts. To achieve this goal it was necessary to process these images, in order to create the dataset, as well as to create data augmentation methods, after that two deep learning algorithms were used, one for object detection and one for semantic segmentation, YOLOv7 and Unet respectively. At the end, we made inferences on the provided images in order to try to find new archaeological sites not yet annotated.}
